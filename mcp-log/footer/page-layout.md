# footer page-layout（コンポーネント単位）

共通コンポーネント `components/_footer.scss`。PC と SP で構成が異なる。

## PC（高さ約166px・bg #333）
- 上段（flex 横1列）: ロゴ｜会社情報＋GoogleMap｜縦罫線｜ナビ(業務/設備/会社/採用 横並び)｜お問い合わせpill｜トップへ戻る丸
- 横罫線（白0.3px）
- 下段（flex 横）: プライバシーポリシー(左) … Copyright(右)

## SP（縦積み・bg #333）
1. ナビ（5行リスト・白丸→・行間40px・各行下罫線）
2. ロゴ（中央）
3. お問い合わせ pill（中央）
4. 会社情報（住所＋Google Map）
5. 横罫線
6. 下段: プライバシーポリシー / Copyright（左）＋ トップへ戻る角（右下 to-top_sp）

## 縦間隔（SP・rel-y より）
- ナビ末(207) → ロゴ(243.16): gap≈36
- ロゴ下(≈297) → ボタン(323.88): gap≈27
- ボタン下(372) → 住所(393.56): gap≈22
- 住所下(≈435) → 罫線(449.18): gap≈14
- 罫線 → プライバシー(464): gap≈15

## 整合性
- ✅ PC/SP とも bg #333333・白文字で一致
- ⚠️ PC インナーは左右非対称（左119/右44）＝to-top が右に張り出す構成。flex space-between で近似。
- ⚠️ ナビは PC=横4項目(TOP無)・SP=縦5項目(TOP有・矢印付) → 同一 ul を CSS で出し分け（TOP/矢印は PC で非表示、PC は inline 横並び）。
