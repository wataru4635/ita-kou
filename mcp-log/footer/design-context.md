## お問い合わせボタン ホバー（高圧フランジ準拠・md以上）
- 通常: 白bg / 文字#003894 / 矢印=青(btn-arrow-blue)
- ホバー: bg→#003894（navy） / 文字→白 / 矢印→白（青SVGに `filter: brightness(0) invert(1)` で白化）
- transition .3s。SP（< md）はホバー無効。

## フォント/ウェイト
- 12px Medium(500)・13px Bold(700)・10px Light(300)/Medium(500)・14.72px Bold(700) → 既存 --font-weight-*（300/500/700）で対応。
- ls: Copyright/SP系 = 40→0.04em、ITAOKA = 100→0.1em。
