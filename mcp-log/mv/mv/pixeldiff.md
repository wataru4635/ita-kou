## 残存差分の内訳と原因
- **主因（初回の低スコア）**: テスト用ブラウザの縦スクロールバー(15px)が実装幅を 1440→1425 / 375→360 に縮め、全体が水平にずれて写り込んでいた。実機（オーバーレイ式スクロールバー）では発生しない計測アーティファクト。撮影時に `overflow:hidden` で除去して正味比較。
- 残差: 縦書きキャッチ「精度に妥協しない。/選ばれるものづくり。」と英字の**フォント字形差**（XD はベクターアウトライン文字・実装は CSS 縦書き Noto Sans JP Bold）。位置・サイズは一致、字形のみ差。
- 背景写真は object-fit:cover + object-position:50% 44% で XD クロップとほぼ一致（PC NCC 0.87）。
