# mv page-layout

トップpage最初のセクション。全画面（PC 100vh / SP 100dvh）の1枚物（スライダー）。
セクション間 margin-top なし（ページ先頭・アートボード上端 y0 から開始）。次セクションとの gap は MV の後続セクション実装時に別途。

## レイヤー（重なり順）
1. スライダー画像（swiper・cover）z0
2. 黒オーバーレイ #000 0.35 z1
3. キャッチ（縦書き白・中央）／英字（白）z2
4. ページネーション（左下）／スクロールダウン（右下）z2
5. ヘッダー（position:absolute・別コンポーネント）が上部に重なる

## 整合性
- ✅ PC/SP とも全画面・黒35%オーバーレイ・白テキスト・3点ページネーション(アクティブ黄)
- ⚠️ キャッチはベクターパス → CSS 縦書きテキストで近似（字形差は許容）
- ⚠️ キャッチ/英字の縦位置は中央やや下（PC: キャッチ中心 rel-y519, 英字中心467）→ 実装後 pixeldiff で微調整
